# Portfolio Website

A sleek, responsive, and modern personal portfolio website built using React, Vite, and Tailwind CSS. It highlights my projects, skills, and technical journey in a clean, high-performance interface.

---

## 🚀 Tech Stack

* **Frontend:** React (Functional Components, Hooks)
* **Build Tool:** Vite (Fast Refresh, Optimized Production Builds)
* **Styling:** Tailwind CSS (Utility-first, Responsive Design)
* **Icons:** Lucide React / React Icons

---

## ✨ Features

* **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens.
* **Dynamic Sections:** Modular architecture including Home, About, Skills, Projects, and Contact components.
* **Performance Optimized:** Near-instant loading times and efficient asset compilation powered by Vite.
* **Clean UI/UX:** Modern typography, smooth transitions, and a clean minimalist aesthetic.

---

## 🛠️ Installation & Setup

Follow these steps to run the project locally:

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/your-portfolio-repo.git
cd your-portfolio-repo

```


2. **Install dependencies:**
```bash
npm install

```


3. **Start the development server:**
```bash
npm run dev

```


4. **Build for production:**
```bash
npm run build

```
